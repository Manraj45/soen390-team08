# soen390-team08

## Description

Bike ERP is the ERP system develeped by Badob Inc. The main features of our product are:
- Planning
- Scheduling
- Transport Planning/Shipping
- Vendors
- Etc.

Our product will allow creating, editing and tracking of bike components.
